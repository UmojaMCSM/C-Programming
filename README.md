### Compilation
- For some programs, you may need to use the -lm flags to ensure linking is successful:
- Example: **gcc my_program.c -o my_exe_name -lm**

### Using Git - The Quick Manual of Common Commands

|**Command**|**Purpose**|
|--|--|
|git status | Check the status of the changes (if any) you've made to your repository.|
|git add <insert file name> | Prepare/Stage files that have been changed for a commit.|
|git commit -m "Your commit message" | Commit files so they're ready to be pushed. Add a message describing the change to be made. |
| git push | Officially push changed files to your repository. |
| git log | Review your commit history. |
